## Engine Dict

A simple dictionary.

### Build dependencies

`sudo apt install g++ qt5-default libdtkbase-dev libdtkwidget libdtkutil-dev`

### Runtime dependencies

* qt5widgets

* qt5gui

* qt5core

`sudo apt install libqt5core5a libqt5gui5 libqt5widgets5 libdtkbase libdtkwidget libdtkutil`
